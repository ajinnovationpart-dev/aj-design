## 🎯 모바일 특화 컴포넌트 발견

### 1. 탭 네비게이션 (Horizontal Tabs)
**위치**: 화면 상단 (헤더 아래)

**구조**:
```
차량정보 | 상차정보 | 경유지 | 하차정보 | 부가정보
   (활성)                    (활성)
```

**특징**:
- 스크롤 가능한 가로 탭
- 활성 탭: 파란색 텍스트 + 하단 border
- 비활성 탭: 회색 텍스트
- 터치 친화적 간격

---

### 2. 칩 버튼 그룹 (Chip Buttons)
**용도**: 단일/다중 선택

**패턴 A: 3개 그리드**
```
┌─────────┬─────────┬─────────┐
│  상온   │  냉장   │  냉동   │
└─────────┴─────────┴─────────┘
```

**패턴 B: 4개 그리드**
```
┌──────┬──────┬──────┬──────┐
│ 카고 │ 탑차 │ 윙바디│ 무관 │
└──────┴──────┴──────┴──────┘
```

**패턴 C: 유동 그리드 (톤수)**
```
┌────┬──────┬──────┬──────┐
│ 1톤│ 1.4톤│ 2.5톤│ 3.5톤│
├────┼──────┼──────┼──────┤
│ 5톤│ 5톤축│ 11톤 │ 18톤 │
├────┼──────┼──────┤
│25톤│ 라보 │ 다마스│
└────┴──────┴──────┘
```

**특징**:
- 선택됨: 파란 테두리 + 파란 텍스트
- 선택 안됨: 회색 테두리 + 검정 텍스트
- 높이: ~48-56px (터치 최적화)
- 간격: 8-12px

---

### 3. 시간 선택 버튼 그룹
**위치**: 하차 날짜 및 시간 섹션

```
[ 24시간 뒤 ] [ 48시간 뒤 ] [ 익일 오전 ] ☑ 시간 엄수
```

**특징**:
- 빠른 선택 옵션
- 체크박스와 조합
- 반응형 버튼 크기

---

### 4. 날짜/시간 인풋 (Combined Picker)
```
┌──────────┬────────────┐  ┌────────┬──────────┐
│  📅     │ 2023.08.01 │  │  🕐    │  16:00   │
└──────────┴────────────┘  └────────┴──────────┘
```

**특징**:
- 아이콘 + 값 표시
- 탭하면 네이티브 피커 오픈
- 2열 그리드 레이아웃

---

### 5. 연락처 선택 칩
```
[ 홍길동 ] [ 이순신 ] [ 김종서 ]

┌──────────────────────────────────┐
│  010-8697-9823                    │
└──────────────────────────────────┘
```

**패턴**:
- 상단: 이름 칩 버튼 (수평 스크롤)
- 하단: 선택된 연락처 표시
- 우측: "연락처 선택 >" 링크

---

### 6. 대형 아이콘 선택 버튼
```
┌──────────────────┬──────────────────┐
│       🚛         │       👤         │
│     지게차       │     수작업       │
└──────────────────┴──────────────────┘
```

**특징**:
- 큰 아이콘 (48-64px)
- 텍스트 라벨
- 2열 그리드
- 선택됨: 파란 테두리

---

### 7. 큰 하단 고정 버튼 (Bottom Fixed Button)
```
┌─────────────────────────────────────┐
│                다음                  │  ← 파란색, 전체 너비
└─────────────────────────────────────┘
```

**특징**:
- 화면 하단 고정
- 전체 너비
- 높이: 56-64px
- Safe Area 고려 (홈 인디케이터)

---

### 8. 입력 필드 (Text Input)
```
┌─────────────────────────────────────┐
│  안녕물류                            │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  인천시 부평구 부흥로 248            │
└─────────────────────────────────────┘
```

**특징**:
- 큰 터치 영역 (48px)
- 명확한 테두리
- 플레이스홀더 또는 값 표시

---

### 9. 섹션 헤더 with 액션
```
하차지 주소 *              하차지 선택 >
```

**패턴**:
- 좌측: 라벨 + 필수 표시(*)
- 우측: 액션 링크 (파란색, >)

---

### 10. Select/Dropdown (모바일)
```
┌──────────────────┬──────────────────┐
│  수량 선택    ▼  │  유형 선택    ▼  │
└──────────────────┴──────────────────┘
```

**특징**:
- 네이티브 select 스타일
- 2열 그리드 배치
- 우측 화살표 아이콘

```javascript
primary: '#5A7CF0'

white: '#FFFFFF'
```

### 간격 (모바일 최적화)
```javascript
xs: '4px'
sm: '8px'
md: '12px'
lg: '16px'
xl: '20px'
xxl: '24px'
```

### 터치 영역
```javascript
최소 높이: 44px (iOS 권장)
권장 높이: 48-56px
버튼 간격: 8-12px
```

### 폰트 크기 (모바일)
```javascript
xs: '11px'    // 보조 텍스트
sm: '12px'    // 작은 라벨
base: '14px'  // 기본 텍스트
md: '15px'    // 입력 필드
lg: '16px'    // 버튼, 헤더
xl: '18px'    // 타이틀
xxl: '20px'   // 페이지 제목
```

### Border Radius (모바일)
```javascript
sm: '6px'     // 칩, 작은 버튼
md: '8px'     // 일반 버튼, 입력
lg: '12px'    // 카드, 큰 버튼
xl: '16px'    // 모달, 바텀시트
```

## 🔍 주요 발견 사항

### 1. 터치 최적화
- 모든 버튼 최소 48px 높이
- 넉넉한 간격 (8-12px)
- 큰 터치 영역

### 2. 간결한 입력
- 칩 버튼으로 빠른 선택
- 시간 프리셋 (24시간 뒤, 48시간 뒤)
- 즐겨찾기 불러오기

### 3. 단계별 입력
- 5개 탭으로 폼 분할
- 하단 "다음" 버튼으로 진행
- 진행 상황 명확

### 4. 모바일 네이티브 활용
- 날짜/시간 선택: 네이티브 피커
- 연락처: 연락처 선택 UI
- 스크롤: 세로/가로 자유

### 5. 시각적 계층
- 아이콘 활용 (지게차, 수작업)
- 색상으로 상태 구분
- 명확한 섹션 구분

## 📱 모바일 vs 데스크톱 차이점

| 항목 | 데스크톱 | 모바일 |
|------|----------|--------|
| **네비게이션** | 좌측 사이드바 | 하단 탭 + 상단 탭 |
| **입력** | Select, Input | 칩 버튼 + 간소화 Input |
| **날짜 선택** | 커스텀 DatePicker | 네이티브 피커 |
| **액션 버튼** | 우측 상단/하단 | 하단 고정 버튼 |
| **선택** | 드롭다운, 체크박스 | 칩 버튼, 큰 터치 버튼 |
| **스크롤** | 주로 세로 | 세로 + 가로 (탭) |
| **간격** | 16-32px | 12-20px |
| **터치 영역** | 마우스 (작아도 됨) | 최소 44-48px |

## 💡 모바일 UX 인사이트

### 1. One-Thumb Operation (한손 사용)
- 주요 액션은 화면 하단 (thumb zone)
- 뒤로가기는 좌측 상단 (reach zone)
- 중요하지 않은 것은 상단 우측

### 2. Progressive Disclosure (점진적 공개)
- 5개 탭으로 정보 분산
- 한 화면에 너무 많은 정보 X
- 다음 버튼으로 단계별 진행

### 3. Smart Defaults (스마트 기본값)
- "24시간 뒤", "48시간 뒤" 버튼
- 자주 쓰는 연락처 칩
- 즐겨찾기 불러오기

### 4. Visual Feedback (시각적 피드백)
- 선택된 칩: 파란 테두리
- 활성 탭: 파란 텍스트 + 밑줄
- 입력 포커스: 테두리 변화

### 5. Touch-Friendly (터치 친화적)
- 모든 터치 영역 48px+
- 버튼 간 최소 8px 간격
- 큰 텍스트, 명확한 아이콘

## 🔥 핵심 패턴 요약

### 1. 칩 버튼 선택 패턴
```jsx
<ChipButtonGroup columns={3}>
  <ChipButton selected>상온</ChipButton>
  <ChipButton>냉장</ChipButton>
  <ChipButton>냉동</ChipButton>
</ChipButtonGroup>
```

### 2. 탭 네비게이션 패턴
```jsx
<HorizontalTabs active={0}>
  <Tab>차량정보</Tab>
  <Tab>상차정보</Tab>
  <Tab>경유지</Tab>
  <Tab>하차정보</Tab>
</HorizontalTabs>
```

### 3. 하단 고정 버튼 패턴
```jsx
<BottomFixedButton onClick={handleNext}>
  다음
</BottomFixedButton>
```

### 4. 섹션 + 액션 패턴
```jsx
<SectionHeader 
  label="하차지 주소"
  required
  action="하차지 선택"
  onAction={handleSelect}
/>
```
