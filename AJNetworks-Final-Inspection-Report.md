## 🆕 추가 발견된 컴포넌트 및 패턴

### 1️⃣ 통계 카드 (Summary Cards) ⭐ 중요
**위치**: 운송내역, 정산내역 페이지 상단

**특징**:
- 4-5개 카드가 나란히 배치
- 카드당: 제목 + 금액 + 변화율
- 색상 코딩 (운송비, 당송료, 배차료, 수수료 등)

**구조**:
```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ 운송비      │ │ 당송료      │ │ 배차료      │ │ 수수료      │
│ 0원         │ │ 5,000,000   │ │ 8,000,000   │ │ 6,000,000   │
└─────────────┘ └─────────────┘ └─────────────┘ └─────────────┘
```

**사용처**:
- 대시보드 요약
- 통계 페이지
- 정산 현황

---

### 2️⃣ 리스트 선택 모달 패턴 ⭐ 중요
**종류**:
- 운송사 선택 모달
- 배차 담당자 선택 모달
- 운송 신청자 선택 모달
- 고객사 선택 모달

**공통 구조**:
```
┌─────────────────────────────────────────┐
│ 제목                              [X]    │
├─────────────────────────────────────────┤
│                          [🔍 검색]       │
├─────────────────────────────────────────┤
│ ⭐ 홍길동    010-1234-5678  [고정해제] [선택]│
│ ⭐ 유재석    010-2345-6789  [고정해제] [선택]│
│ ☆ 양세찬    010-3456-7890  [고정하기] [선택]│
│ ☆ 김호중    010-4567-8901  [고정하기] [선택]│
├─────────────────────────────────────────┤
│         ◀ [1] 2 3 4 5 ▶                 │
└─────────────────────────────────────────┘
```

**핵심 기능**:
- 즐겨찾기 (별표 토글)
- 고정/해제 버튼
- 검색 기능
- 페이지네이션
- 개별 선택 버튼

---

### 3️⃣ 복합 필터 섹션 ⭐ 중요
**위치**: 모든 목록 페이지 상단

**구조**:
```
┌───────────────────────────────────────────────────────┐
│ [운송구분 ▼] [상태 ▼]  [2022/9 ▼]  [운송일]          │
│ [전국 ▼]     [전체 ▼]  [화물명]  [차량 ▼]  [차종 ▼]  │
│ [장비]                                      [검색]    │
└───────────────────────────────────────────────────────┘
```

**컴포넌트 조합**:
- 여러 개의 Select (3-8개)
- DatePicker 또는 Month Picker
- Input 필드 (텍스트 검색)
- 검색 버튼

---

### 4️⃣ 액션 버튼 그룹 ⭐ 중요
**위치**: 테이블 상단 우측

**패턴**:
```
[ + 등록 ]  [ 📥 내보내기 ]  [ 검색초기화 ▼ ]
```

**버튼 종류**:
- 등록 (Primary)
- 내보내기 (Secondary with icon)
- 검색초기화 (Secondary)
- 25개씩 보기 ▼ (Select)

---

### 5️⃣ 고급 데이터 테이블
**특징**:
- 10-15개 컬럼
- 고정 컬럼 (좌측)
- 다중 액션 버튼
- 상태 뱃지
- 정렬 가능 헤더
- 행 호버 효과
- 체크박스 선택

**컬럼 타입**:
- 텍스트
- 날짜/시간
- 금액 (숫자 + 원)
- 상태 (뱃지)
- 액션 (버튼)

---

### 6️⃣ 상세 정보 섹션
**패턴**: 폼 형태의 정보 표시

```
┌─────────────────────────────────────┐
│ 기본 정보                            │
├─────────────────────────────────────┤
│ 운송구분    화물                     │
│ 차량번호    12가 3456                │
│ 차종        윙바디                   │
├─────────────────────────────────────┤
│ 상차지 정보                          │
├─────────────────────────────────────┤
│ 주소        서울시 강남구...         │
│ 연락처      010-1234-5678            │
└─────────────────────────────────────┘
```

---

### 7️⃣ 탭 그룹 패턴
**종류**:
- 고정해제 / 고정하기 (토글 탭)
- 상차지 / 하차지 / 경유지
- 매입 / 매출

**스타일**:
```
┌────────┬────────┐
│ 고정해제 │ 고정하기│  ← 하단 border로 active 표시
└────────┴────────┘
```

## 🎨 추가로 구현이 필요한 컴포넌트

### 1. SummaryCard (통계 카드)
```jsx
<SummaryCard
  title="운송비"
  value="5,000,000"
  unit="원"
  color="primary"
/>
```

### 2. FilterSection (필터 영역)
```jsx
<FilterSection>
  <Select label="운송구분" />
  <Select label="상태" />
  <DatePicker label="운송일" />
  <Button>검색</Button>
</FilterSection>
```

### 3. ListModal (리스트 선택 모달)
```jsx
<ListModal
  title="운송사 선택"
  items={carriers}
  onSelect={handleSelect}
  favoritable={true}
  pinnedable={true}
/>
```

### 4. ActionButtonGroup (액션 버튼 그룹)
```jsx
<ActionButtonGroup>
  <Button icon={<Plus />}>등록</Button>
  <Button icon={<Download />}>내보내기</Button>
  <Select options={pageSizes} />
</ActionButtonGroup>
```

### 5. AdvancedTable (고급 테이블)
```jsx
<AdvancedTable
  columns={columns}
  data={data}
  selectable={true}
  sortable={true}
  fixed={['checkbox', 'id']}
/>
```

### 6. InfoSection (정보 섹션)
```jsx
<InfoSection title="기본 정보">
  <InfoRow label="운송구분" value="화물" />
  <InfoRow label="차량번호" value="12가 3456" />
</InfoSection>
```

### 7. EmptyState (빈 상태)
```jsx
<EmptyState
  icon={<FileText />}
  message="공통 테이블 내역 없음"
/>
```

## 🔍 주요 발견 사항

### 1. 일관된 디자인 시스템
- 모든 페이지가 동일한 컴포넌트 사용
- 색상, 간격, 타이포그래피 일관성 유지
- 예측 가능한 사용자 경험

### 2. 데이터 중심 설계
- 복잡한 테이블 구조
- 다양한 필터 옵션
- 통계 정보 강조

### 3. 모바일 미지원
- 데스크톱 중심 디자인
- 넓은 테이블 (1200px+)
- 다중 컬럼 레이아웃

### 4. 색상 사용
- **Primary (#5A7CF0)**: 주요 액션
- **Brand (#E31E24)**: 브랜드 강조
- **Success (#10B981)**: 완료 상태
- **Warning (#F59E0B)**: 대기/진행
- **Gray**: 텍스트 계층
